#Десктоп Мимиск для эффективного управления задачами в распределённой среде формируемой ESMP и Jira
Приложение работает в режиме приложения рабочего стола, открываясь поверх других приложений. 
Позволяет:
хранить в одном месте часто упоминаемые jira, включая возможность их выделения, копирования и прямого перехода по ним без указания ссылки; 
планировать работу над заявками ESMP, устанавливая необходимый статус, приоритет и имея возможность откладывать заявки; 
хранить в одном месте пароли и скрипты для ответов в удобном для копирования виде, для скриптов есть возможность указания текущего времени. 
## Запуск
npm install
npm run dev

## Сборка
Поддержка только для Windows
npm install
npm run dist
В результате в папке release появится exe-файл.

## Конфигурация
В папке jsonData определить свои json-объекты по аналогии с имеющимися.
При необходимости коррекции числа строк или добавлении новых перейдите в папку components в соответствующий Manager и определите новую с новыми json-данными.
